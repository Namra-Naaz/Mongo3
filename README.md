# Mongo3 Project

This is a Node.js project using MongoDB and Express.

## How to Run

1. Install dependencies:
> npm init -y (package.json)
> npm i express
> npm i ejs
> npm i mongoose

2. Start the app:
create file
> touch index.js
> initialize server (nodemon index.js)

## Features

- Express server
- MongoDB connection
- Organized folder structure
